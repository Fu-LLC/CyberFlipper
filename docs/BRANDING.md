# CyberFlipper Branding & Customization

## SD Card Payloads
- All payloads should start with a CyberFlipper comment header.
- Example:
  ```
  REM CYBERFLIPPER v1.2.0 - Custom Payload
  REM FOR AUTHORIZED USE ONLY
  ```

## Web Dashboard
- Use the provided CSS/JS for consistent theming.
- Add your logo to `assets/` and reference in `index.html`.
